Tax Calculator ACRF 

== Installation ==

1. Upload `acrf-tax-calculator.zip` to the `/wp-content/plugins/` directory
2. Unzip the file
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Add the shortcode [tax_calculator_shortcode] to an existent page
5. To edit the title and subtitle, add attributes to the shortcode
    e.g.
    [tax_calculator_shortcode title="Your new title" subtitle="Your new subtitle"]
